  ## Installation
  
  To install necessary dependencies, run the following command:
  
  ```bash

  npm i

  ```
  
  ## Usage
  
  Run this in the terminal to generate a readme file!
  
  To run tests, run the following command:
  
  ```bash
  npm test
  ```
